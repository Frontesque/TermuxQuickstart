Install [Stable]:
```
pkg install wget -y && wget https://raw.githubusercontent.com/Frontesque/TermuxQuickstart/master/TermuxQuickstart.sh && chmod +x TermuxQuickstart.sh && bash TermuxQuickstart.sh
```

Install [Development]:
```
pkg install git -y && git clone --branch development https://github.com/Frontesque/TermuxQuickstart && cd TermuxQuickstart && chmod +x TermuxQuickstart.sh && bash QuickstartLauncher.sh
```
