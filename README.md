Install [Stable]:
```
pkg install wget -y && wget https://raw.githubusercontent.com/FrontlineMist57/TermuxQuickstart/master/TermuxQuickstart.sh && chmod +x TermuxQuickstart.sh && bash TermuxQuickstart.sh
```

Install [Development]:
```
pkg install git -y && git clone https://github.com/FrontlineMist57/TermuxQuickstart/tree/development && chmod +x TermuxQuickstart.sh && bash QuickstartLauncher.sh
```
